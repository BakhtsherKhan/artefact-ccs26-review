# Anonymous Artifact for Review

## Overview

This artifact supports the evaluation of a paper on delivery integrity in a server-mediated messaging system. It contains representative packet traces, sanitized documentation, and CryptoVerif models used to support the empirical and formal claims in the paper.

The artifact is prepared for double-blind review. It does not contain author names, affiliations, personal GitHub links, private account identifiers, authentication material, or unsanitized device logs.
